Main Menu

Requires the latest SmartPoi firmware running on the SmartPoi hardware.

Controls

Options include Pattern, Speed, and Brightness change. Speed is how long each image is displayed, ranging from 0 to 1800 seconds.

documentation

Fetch and Display

Fetch and display images from the SmartPoi hardware. This section allows you to view the main and auxiliary POI images by fetching them from the Poi file system. Also allows you to upload images to the POI, using convenient drag and drop, as well as replace or delete images on the POI.

documentation

List

Check the files on the POI by opening this section while connected to the POI (AP). Click on "Get File List" to view the files.

documentation

Upload Bin

Upload multiple backup images to SmartPoi for offline display. Images are resized, rotated, and converted to .bin format before being uploaded sequentially to the two POI.
